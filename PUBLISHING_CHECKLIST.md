## How to publish on GitHub

1. Create a GitHub account or sign in.
2. Select **New repository**.
3. Use the repository name `cosy-watch-octopus-heat-pump-widget`.
4. Add the description above.
5. Set the repository to **Public**.
6. Do not tick the box to add a README, licence or gitignore because these files are already included.
7. Create the repository.
8. Upload the files from this folder.
9. Commit the files.
10. Go to **Settings** → **Pages**.
11. Under **Build and deployment**, choose **Deploy from a branch**.
12. Choose the `main` branch and the `/docs` folder.
13. Save the GitHub Pages settings.

## Before sharing publicly

- Add your screenshots to the `screenshots` folder.
- Rename them as `normal-operation.png` and `fault-state.png`, or update the README links.
- Check the script still says `PASTE_YOUR_API_KEY_HERE` and does not contain your real API key.
- Add the GitHub Pages link to the repository About section.
