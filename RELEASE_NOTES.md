# Release Notes

## Version 1.0.0

Initial public release of Cosy Watch.

### Included in this release

- Medium iPhone Home Screen widget for Octopus Cosy heat pumps
- Small widget layout
- Hot water temperature display
- Cosy Pod temperature readings
- Average room temperature across multiple pods
- Temperature range across multiple pods
- Average humidity across pods
- Heat pump status display
- Fault warning display with amber highlighting
- Fault code display where available
- Live performance data where available
- Current power input and heat output
- COP display where available
- Today’s energy use
- Yesterday’s energy use
- Outdoor temperature
- Automatic heat pump discovery using the Octopus account
- Saved fallback readings if the live API request fails
- Optional custom pod names
- Optional tap-through link to the Octopus dashboard
- Test modes for pod and fault previews

### Notes

This release was built and tested on an Octopus Cosy 9. It uses Octopus API endpoints that may change in future.
