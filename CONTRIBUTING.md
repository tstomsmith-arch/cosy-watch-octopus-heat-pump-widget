# Contributing

Thanks for your interest in improving Cosy Watch.

This project is intended to stay simple and useful for everyday Octopus Cosy users, including people who have little or no coding experience.

## Good contributions

Helpful changes could include:

- Clearer setup instructions
- Bug fixes
- Better handling of missing API data
- Improved support for different Cosy setups
- Cleaner widget layout options
- Better fault display
- More screenshots
- Better troubleshooting notes

## Before sharing code

Please check that you have not included:

- Your Octopus API key
- Your account number, unless you deliberately chose to share it
- Your heat pump controller ID
- Screenshots showing private account information

## Reporting a problem

If you raise an issue, it helps to include:

- iPhone model
- iOS version
- Scriptable version
- Heat pump model, if known
- Whether you are using the small or medium widget
- What the widget displayed
- Any Scriptable error message

Please do not include your API key.

## Pull requests

If you make a pull request:

1. Keep the code readable
2. Add comments where the logic may not be obvious
3. Avoid adding unnecessary dependencies
4. Update the README if the setup steps change
5. Keep privacy and API key safety in mind

## Project tone

Please keep discussion friendly and practical. This is a community project for people trying to make their heat pump data easier to see.
